## awscli 설치하기 

pip 을 사용해서 설치한다 

```
pip install awscli

```


## eb cli 설치하기 

```
brew update
brew install awsebcli
```

## eb cli 명령어

elastic beanstalk 상태 보기 
```
eb status
```
